# Test-task

Личный кабинет с профилями пациентов.

## Архитектура

Feature Sliced Design

## Технологии

- React (Next.js)
- TypeScript
- Препроцессор SCSS

## Запуск проекта

- Клонировать репозиторий ```git clone```
- Установить зависимости ```npm install```
- Собрать проект ```npm run build```
- Запустить проект ```npm start```
